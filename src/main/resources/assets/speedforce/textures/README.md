# Speed Force Mod - Textures

This folder should contain texture files for the mod.

## Required Textures

### Block Textures
- `textures/block/particle_accelerator.png` - 16x16 or 32x32 PNG

### Item Textures  
- `textures/item/flash_helmet.png` - 16x16 PNG
- `textures/item/flash_chestplate.png` - 16x16 PNG
- `textures/item/flash_leggings.png` - 16x16 PNG
- `textures/item/flash_boots.png` - 16x16 PNG

## Notes
- If textures are missing, Minecraft will display the default missing texture (purple-black checkerboard)
- You can create temporary placeholder textures or use existing Minecraft textures for testing
